# 🤖 2BPM Bot Discord

Bot desenvolvido para automação e gerenciamento de servidores no Discord, com foco em **organização, escalabilidade e experiência do usuário**.

> Projeto voltado à prática de desenvolvimento backend, arquitetura modular e integração com APIs.

---

##  Destaques do Projeto

- Estrutura modular (código organizado e escalável)
- Automação de tarefas administrativas
- Controle de permissões por cargo
- Registro e rastreamento de ações
- Foco em usabilidade (comandos simples e intuitivos)
